## Goal

Create a **`TodoProvider`** class extending **`ChangeNotifier`** that loads todos from JSONPlaceholder and holds **`List`**, **`loading`**, and **`error`**.

## Steps

1. Add **`provider`** to `pubspec.yaml`
2. Branch: `week3-group-5/todo-provider`
3. Copy ideas from `sample-code/todo_provider.dart` — adapt names if needed
4. Implement **`loadTodos()`** with `notifyListeners()` after state changes
5. **Do not** wire UI yet — you can test with `debugPrint` in provider if needed
6. Commit, push, PR
