# Group 9 — Form input

**Branch:** `week3-group-9/search-field`

---

## Goal

Add a **`TextField`** (search or filter) that **filters** the todo list **in memory** as the user types (no new API call required).

---

## Steps

1. Branch: `week3-group-9/search-field`
2. Add **`TextEditingController`** or **`onChanged`** on **`TextField`**
3. Keep full list in provider or state; compute **filtered** list for **`ListView`**
4. Case-insensitive **contains** is enough
5. Commit, push, PR

---

## Done when

Typing narrows the visible todos.
